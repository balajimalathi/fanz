## What is RepoVox?
RepoVox helps indie hackers, solo-developers, teams, and project managers **automate GitHub insights** into summaries and code review report for **Twitter, Slack and Mail**. Stay updated with project activities and share your contributions effortlessly.

## Features in v0.1.0
- **GitHub OAuth Integration** - Connect your GitHub account securely.
- **Automated Content Generation** - Convert PR merges, pushes, and commits into shareable posts.
- **Multi-Platform Posting** - Publish directly to Twitter, Slack and Mail.
- **Customizable Triggers** - Define what activities should be converted into content.
- **Repo Activity Dashboard** - Track insights and generated content in one place.
- **Slack Integration** - Get updates in Slack channels.
